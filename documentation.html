Documentation for Culinary Prefrence Organizer

Product Scope

The Culinary Preference Organizer is a web application designed to help users organize their culinary preferences, including favorite recipes, dietary restrictions, and cooking preferences.

Its purpose is to provide users with a centralized platform to manage their culinary interests and recommend recipes.

By using this web application, users can benefit from:

  • Organization
  • Personalization
  • Discoverability
  • Efficiency

Product functions

The Culinary Preference Organizer provides the following functions:

  • Sign up
  • Log in
  • Log out
  • View/Edit profile
  • Search recipes
  • View/Add/Edit/Delete dietary restrictions
  • View/Add/Edit/Delete cooking preferences
  • Create/Edit/View shopping list

External Interface Requirements

This section describes the various interfaces required by the web application for optimal operation:

  • User Interfaces: The application features a responsive web design and utilizes standardized GUI elements such as buttons and error messages to ensure a user-friendly experience across various devices.
  • Hardware Interfaces: There are no hardware interfaces since this is a purely web-based application.
  • Software Interfaces:
    • API: The application integrates with the Open Food Facts API to retrieve data.
    • Database: PostgreSQL is used for storing user data and preferences, ensuring robust data management and retrieval.
  • Communications Interfaces: The application communicates over the internet using HTTP/HTTPS protocols, adhering to RESTful API standards for data interaction.

System Features

The web application is equipped with several critical features to enhance user experience:

  • User Account Management: Essential for a personalized experience, this high-priority feature allows users to register, log in, and manage their accounts securely. The system ensures robust authentication and profile management capabilities.
  • Shopping List Generator: A core functionality that has a medium priority, enabling users to generate, edit, and store shopping lists based on their preferences and dietary needs.
